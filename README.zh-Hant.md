# 以太坊巨鯨是聰明錢嗎？鏈上訊號與市場情緒的事件研究

## 摘要

本研究探討大額以太坊(Ethereum)鏈上交易是否能預測短期與中期的 ETH 價格走勢,以及市場情緒是否會調節這層關係。我們採用事件研究法(event study methodology),分析 **3.5 年**(2023 年 1 月至 2026 年 7 月)內共 **646,442 筆巨鯨交易**,衡量巨鯨存入交易所(賣出訊號)與提領(買入訊號)的方向命中率,涵蓋九種時間跨度(1 小時至 6 個月)、四種交易規模門檻(100 萬美元至 1,000 萬美元以上)、七種情緒區間、兩種市場狀態(牛市/熊市),以及四個獨立的年度樣本。

**主要發現:**

存入(賣出)訊號的優勢隨時間跨度單調成長,從 24 小時的 +1.3% 成長到 6 個月的 +12.4%,而這個模式是用 2023-2025 年的資料建立的,並在事後蒐集的 2026 年資料中獲得確認,結果不變。

1. **巨鯨存入(賣出訊號)展現出持續且隨時間跨度增強的優勢,且在每一種交易規模下都成立。** 無條件的存入優勢從 24 小時的 +1.3% 成長到 1 個月的 +4.8%,再到 6 個月的 +12.4%。無論是單獨看 100 萬美元以上或 1,000 萬美元以上的存入,這種隨時間成長的趨勢都一致存在:兩者皆從 2023 年大致持平,成長到 2026 年的 +2.9% 至 +3.9%。巨鯨賣家的思考尺度是以週、月計算,而非小時。明確以情緒(極度貪婪)作為條件,只在 24 小時這個時間跨度有幫助,依交易規模不同,可將優勢提升 1.3 至 2.2 個百分點;而在所測試的每一個更長時間跨度下,同樣的條件化反而會削弱優勢,在 6 個月時最多削弱達 17.3 個百分點,且在全部四種門檻下皆一致如此。真正禁得起時間考驗的,是這個無條件訊號本身,而非經情緒篩選後的版本。(我們最初報告的一個較狹窄的獨立主張——極度貪婪期間 1,000 萬美元以上的存入命中率達 78.3%——在進一步檢驗後並未站得住腳;詳見討論章節。)

2. **巨鯨提領(買入訊號)的優勢已完全消失,且在每一種交易規模下皆是如此。** 2023-2024 年間,負資金費率期間的提領展現出 +4.7% 至 +10.1% 的優勢;到了 2025 年,這個優勢在每個門檻下都已趨近於零;到了 2026 年,從 100 萬美元以上到 1,000 萬美元以上的每個門檻都轉為負值,排除了「只是小額、較不精明的參與者稀釋了訊號」這種簡單解釋。在更長的時間跨度下,提領訊號甚至變成主動誤導:1 個月時優勢為 -6.1%,6 個月時為 -12.9%。我們將此歸因於 DeFi 的成熟化:提領行為愈來愈多代表質押、提供流動性與 L2 橋接,而非方向性的買入。

3. **優勢的衰減具有不對稱性,且一個原本應該削弱存入訊號的逆風並未發生。** 隨著 ETH 價格上漲,固定 100 萬美元的門檻所捕捉到的賣家規模與投入程度逐漸縮小(2023 年約 833 ETH,2026 年僅約 250 ETH),這種稀釋效應理應不利於存入優勢。然而優勢反而增強,這與單純的「組成效應」解釋相悖。關於存入訊號為何沒有像提領訊號一樣被套利消除,其中一個可能的解釋是:巨鯨監控工具放大看漲活動的程度,遠高於看跌活動。

4. **第二種獨立方法得出相容的結論。** 一個經前向驗證(walk-forward validated)的隨機森林(Random Forest)模型(24 折,非單次擬合)發現在短時間跨度下存在適度但真實的優勢(+2.0% 至 +2.9%),而其特徵重要性排名將巨鯨相關特徵列在末位,價格動能與市場情緒才是主導因素。這項完全獨立採用的不同技術,得出與事件研究法一致的結論:巨鯨活動是真實但有條件的訊號,而非獨立的預測因子。

5. **這個優勢是真實的,但伴隨著真實的代價。** 針對最終獲利的長時間跨度存入訊號,我們測量了最大不利變動(maximum adverse excursion):也就是訊號生效之前,最糟的未實現虧損。這個數值隨時間跨度急劇增加,從 1 週的 2.7% 到 6 個月的平均 20.1%,其中最差的 10% 的「正確」6 個月交易,在獲利之前曾經歷 54.4% 的不利價格變動。這個優勢並非免費的午餐;要收割它,意味著必須撐過大多數交易者無法忍受的回撤。

<a id="background"></a>
## 背景

在傳統股票市場中,機構持倉是透過每季申報的 13F 文件揭露的,且有明顯的時間落差。相較之下,以太坊等公開區塊鏈的每一筆交易都能在近乎即時的情況下被觀察到。這為研究大戶(large-holder)的即時行為創造了獨特的機會。

過去的「巨鯨監控」研究多半將其視為價格預測問題:把巨鯨資料餵進機器學習模型,嘗試預測報酬率。這種做法混雜了多種訊號,鮮少能單獨分離出巨鯨本身的貢獻。本研究的新做法在三個方面有所不同:

1. **事件研究法**:我們直接衡量巨鯨行為是否能預測價格方向,而非建構一個黑箱預測模型。
2. **情緒條件化**:我們檢驗市場狀態(恐懼與貪婪指數、期貨資金費率)是否會調節巨鯨訊號。
3. **年度穩定性分析。** 我們獨立檢驗每一年(2023、2024、2025、2026)的資料,以偵測優勢的衰減。2026 年作為對 2023 至 2025 年資料所建立假說的樣本外(out-of-sample)檢驗。

<a id="original-contributions"></a>
## 原創貢獻

1. **不對稱的優勢衰減。** 巨鯨買入訊號從 2023 年到 2026 年逐漸衰退,而賣出訊號卻愈趨強勁。我們推測部分原因是巨鯨監控工具傳播看漲活動的程度高於看跌活動,但這個成因尚未獲得確切證實。

2. **長時間跨度的巨鯨智慧。** 巨鯨存入優勢隨時間跨度單調成長:24 小時 +1.3%,1 個月 +4.8%,6 個月 +12.4%。時間跨度愈長的數字,事件之間重疊的程度愈高,因此比 24 小時的數字更需要謹慎解讀(詳見限制章節)。巨鯨賣家並非單純的當沖交易者,他們可能擁有更豐富的經驗與知識,能提前數週至數月察覺結構性的轉變。

3. **門檻敏感度,經修正後的結論。** 我們最初報告,極度貪婪期間 1,000 萬美元以上的存入在 24 小時命中率達到 78.3%。深入檢視後發現,這個數字背後的交易集中在少數幾天內,這正是限制章節所描述的觀測值重疊問題,因此我們不將這個數字視為已獲確認的結果(詳見討論章節)。

4. **DeFi 稀釋假說。** 由於 DeFi 的成熟化,提領行為可能代表質押、提供流動性與 L2 橋接,這些都不具方向性。這稀釋了「提領=買入」的假設。然而,存入仍是乾淨的賣出訊號。

5. **樣本外檢驗。** 所有假說皆基於 2023 至 2025 年的資料建立。2026 年資料隨後獨立蒐集,並在未經任何參數調整的情況下進行檢驗。存入訊號通過了檢驗;提領訊號則沒有。

<a id="methodology"></a>
## 研究方法

### 資料處理流程(第一階段)
- 透過 Dune Analytics SQL 查詢取得 646,442 筆大額 ETH 交易
- 合併 [brianleect/etherscan-labels](https://github.com/brianleect/etherscan-labels) 與 [dawsbot/eth-labels](https://github.com/dawsbot/eth-labels) 兩個開源專案,標記出 52,768 個錢包地址
- 標籤覆蓋率:62.8% 的交易至少有一方地址可被識別
- 標記出疑似 MEV 機器人的交易,以供敏感度分析使用

### 交易分類(第二階段)
- 對已知錢包採用規則式標記(交易所存入、提領、DeFi、錢包對錢包)
- 最初建構了一個機器學習分類器(隨機森林),用以預測未知錢包的類別,在以時間切分的保留測試集上達到 71% 準確率。在將標籤資料集從 30 個地址擴增至 52,768 個地址後,標籤覆蓋率提升至 62.8%,降低了對分類器的依賴。該分類器目前仍用於剩餘約 37% 收發雙方皆為未知地址的交易。這與 Harlev 等人(2018)針對比特幣實體類型(10 個類別,約 2 億筆交易)所處理的任務相同:他們的最佳結果是以梯度提升(Gradient Boosting,而非隨機森林)達到 77% 準確率,因此本專案的 71% 與已發表結果處於同一區間,但並非完全相同

類別分佈:

| 類別 | 筆數 | 佔比 | 意義 |
|----------|-------|-------|---------|
| 錢包對錢包 (wallet_to_wallet) | 321,257 | 49.7% | 收發雙方皆非交易所、非 DeFi 地址。可能是場外交易(OTC)、冷錢包轉移,或個人錢包間的資金調度。由於意圖不明,不具方向性訊號,因此在事件研究中被排除 |
| 交易所存入 (exchange_deposit) | 181,105 | 28.0% | ETH 匯入已知交易所。視為賣出訊號。存入交易所的主要原因就是想賣出 |
| 交易所提領 (exchange_withdrawal) | 124,772 | 19.3% | ETH 從已知交易所提領。過去被視為買入訊號(累積部位),但隨著 DeFi 成熟,這個假設已逐漸弱化 |
| DeFi 互動 (defi_interaction) | 19,308 | 3.0% | 其中一方地址為已知 DeFi 協議(如 Uniswap、Aave 等)。視為部署資金(看漲訊號)。顯示出些微優勢(52% 命中率),但樣本數較小 |

### 情緒資料(第三階段)
- **恐懼與貪婪指數:** 每日綜合指數 0-100(波動度、成交量、社群媒體、比特幣主導率)
- **Binance 資金費率:** 每 8 小時更新一次的真實資金情緒指標。正值 = 多方支付空方(市場偏多);負值 = 空方支付多方(市場偏空)。Alexander 等人(2020)確立了這背後所依據的永續合約價格發現微觀結構基礎
- **新聞情緒:** 5,906 篇比特幣相關文章以 VADER 進行情緒評分。經測試後發現對價格預測沒有幫助,已捨棄不用。市場衍生的情緒指標(恐懼與貪婪指數、資金費率)有用得多。新聞情緒的程式碼仍保留在儲存庫中,但未用於核心分析
- **巨鯨行為是否跟隨新聞？** 另一項獨立檢驗:在新聞資料與巨鯨資料重疊的約 20 個月期間(2023 年 1 月至 2024 年 9 月,共 484 天至少有一篇文章),每日新聞情緒與每日巨鯨淨流量(存入金額減提領金額)之間並無關聯。相關係數幾乎為零(r=+0.06,不顯著),而看跌新聞日與看漲新聞日的巨鯨平均活動量幾乎相同(p=0.95)。巨鯨既不是在確認,也不是在反駁公開的新聞論述;他們似乎獨立於新聞行事。這是本專案中新聞情緒的第二個空結果:前一項發現它無法預測價格,而這項檢驗發現它也無法反映巨鯨行為(詳見 `scripts/run_sentiment_whale_consistency.py`)。這個解讀與雜訊交易者理論(noise-trader theory,De Long 等人,1990;Black,1986;Shleifer & Summers,1990)一致,該理論將根據私有或鏈上資訊行動的知情交易者,與依據公開情緒反應的雜訊交易者區分開來;不過這項實證檢驗本身(巨鯨流量與 VADER 情緒評分新聞的比較)是本專案自行設計的,並非取自文獻
- **關於恐懼與貪婪指數本身的提醒:** 文獻中關於 CFGI 是否能預測加密貨幣報酬的證據其實相當分歧。部分研究發現兩者存在關聯(Łęt 等人,2023;Cavalheiro 等人,2024);而一項較新的向量自迴歸(VAR)研究則發現因果方向恰好相反——是比特幣報酬帶動指數變化,而非反過來,且情緒指標並未帶來任何樣本外的預測增益。本專案僅將恐懼與貪婪指數作為條件變數使用(第 2 節),從未將其視為獨立的預測因子,而這正是這些分歧證據實際支持的立場

### 事件研究(第四階段——核心分析)
- 最接近的學術類比:Hoang 與 Baur(2022)發現比特幣交易所儲備的流入/流出與同期及未來報酬呈負相關(使用的是彙總儲備資料);Ante 與 Fiedler(2021)則針對大額比特幣轉帳進行了逐筆交易的事件研究。兩者皆針對比特幣,分別採用彙總儲備或轉帳層級的研究設計;本專案針對以太坊、逐筆交易的設計屬於同一研究傳統,但文獻中尚未有直接的複現研究(詳見[相關學術文獻](#related-academic-literature))
- 針對每一筆巨鯨交易,計算其後 1 小時、6 小時、24 小時、3 天、1 週、2 週、1 個月、3 個月、6 個月的 ETH 遠期報酬
- 衡量命中率:巨鯨的行動是否正確預測了價格方向？
- 與**基準率**(同一市場條件下所有時段)比較,以分離出巨鯨特有的優勢
- 年度穩定性分析:獨立分析 2023、2024、2025 年,2026 年作為樣本外檢驗
- 門檻敏感度:100 萬、200 萬、500 萬、1,000 萬美元最低交易規模
- 透過二項式檢定(binomial test)檢驗統計顯著性

<a id="1-unconditional-hit-rates"></a>
### 1. 無條件命中率

完整資料集,646,442 筆交易。命中率 > 50% = 聰明錢,= 50% = 隨機。

| 行為 | 檢驗方向 | 1 小時 | 6 小時 | 24 小時 |
|--------|-----------------|-----|-----|------|
| 交易所存入 | 價格是否下跌？ | 49.5% | 49.3% | **50.5%*** |
| 交易所提領 | 價格是否上漲？ | **50.5%*** | 50.3% | 49.7%* |
| DeFi 互動 | 價格是否上漲？ | **50.9%*** | **51.1%*** | **52.0%*** |

\* p < 0.05, \*\* p < 0.01, \*\*\* p < 0.001

整體而言,巨鯨的表現並不比丟硬幣好多少。優勢是有條件的。

<a id="8-drawdown-during-the-holding-period"></a>
### 8. 持有期間的回撤

上述長時間跨度的結果,僅回報持有期間結束時的報酬。一筆最終獲利 +5% 的部位,可能在過程中曾經跌到 -20%,這是大多數交易者無法忍受的,即使訊號「最終」是正確的。本節直接衡量這個現象:最大不利變動(maximum adverse excursion, MAE),也就是遵循存入(賣出)訊號的交易者,在最終結果出爐之前,按市值計價(mark-to-market)所曾經歷的最糟未實現虧損,這是根據持有期間內完整的每小時價格路徑計算的,而非僅看終點。

下表僅涵蓋最終正確的存入訊號(即在 t+h 時價格確實較低),但關注的問題是:在獲勝之前,需要承受多少痛苦？

#### 巨鯨存入(賣出訊號),無條件——最終正確交易的 MAE

| 時間跨度 | N | 平均 MAE | 中位數 MAE | P90 MAE | 平均最終報酬 |
|---------|---|----------|-------------|---------|---------------------|
| 1 週 | 91,764 | 2.7% | 2.0% | 6.2% | -6.4% |
| 2 週 | 95,088 | 4.3% | 3.5% | 9.7% | -8.8% |
| 1 個月 | 99,620 | 6.0% | 5.2% | 12.9% | -12.8% |
| 3 個月 | 93,568 | 9.6% | 7.6% | 21.0% | -26.1% |
| 6 個月 | 92,904 | **20.1%** | 13.3% | **54.4%** | -30.4% |

#### 巨鯨存入(賣出訊號),條件:極度貪婪(FnG > 75)——最終正確交易的 MAE

| 時間跨度 | N | 平均 MAE | 中位數 MAE | P90 MAE | 平均最終報酬 |
|---------|---|----------|-------------|---------|---------------------|
| 1 週 | 10,760 | 2.8% | 2.3% | 6.0% | -8.0% |
| 2 週 | 11,723 | 4.4% | 3.8% | 10.3% | -9.3% |
| 1 個月 | 15,707 | 6.3% | 4.9% | 13.7% | -12.0% |
| 3 個月 | 17,890 | 12.3% | 9.7% | 29.4% | -27.0% |
| 6 個月 | 19,539 | **15.5%** | 12.4% | 31.7% | -27.6% |

MAE 隨時間跨度增加,增幅大致與優勢本身同步。一個最終獲利的 6 個月存入訊號(無條件情況),平均需要承受 20.1% 的不利價格變動,而最差的 10% 的「正確」交易,曾經歷價格反向上漲 54.4%。第 5 節記錄的優勢是真實的,但交易者(或停損規則)必須撐過這種規模的回撤才能真正收割它。訊號最終錯誤的交易,兩項指標都更糟,例如無條件的 6 個月錯誤交易平均 MAE 高達 80.1%(完整明細見 `results/drawdown_analysis.csv`)。

這裡並未模擬實際的停損規則。MAE 只告訴我們曾經到達的最糟點位,並不代表某個具體的風險管理規則是否能夠撐過去。模擬一個具體的停損政策是下一步的工作。

MAE 這個名稱本身是一個業界從業者的概念(Sweeney,1997),而非學術概念:沒有任何同行評審期刊論文正式採用這個術語。但其背後的核心概念——持有期間內、與路徑相關的回撤——確實有嚴謹的學術基礎(Grossman & Zhou,1993;Magdon-Ismail 等人,2004;Chekhlov、Uryasev & Zabarankin,2005),就算本節沿用 Sweeney 最初提出的名稱,這些文獻仍是該概念更具學術辯護力的引用來源。

<a id="9-bull-vs-bear-market-regimes"></a>
### 9. 牛市與熊市的市場狀態

對 78.3% 這個主張的調查(詳見討論章節)顯示了它為何站不住腳:背後的交易集中在僅僅 10 個日曆天內,這些交易在 24 小時時間跨度下高度重疊,並在長時間跨度下完全主導了結果。這項調查得到的主要教訓——現有資料無法信任長時間跨度的結果——啟發了另一項獨立實驗的構想:比較巨鯨訊號在牛市與熊市之間的差異。結果發現,這項實驗恰好只需要短到中期的時間跨度(24 小時、3 天、1 週)就能得出有意義的結果,這與 78.3% 調查已經教會我們該信任哪些結果相吻合。本節採用第 4 節相同的 100 萬至 1,000 萬美元以上門檻範圍進行檢驗:巨鯨訊號在牛市與熊市之間是否有差異,而這個差異是否在每一種交易規模下都成立？

市場狀態定義:採用標準教科書定義,並未針對資料進行調整。當價格從最近高點回落 20% 時,即進入熊市;當價格從最近低點反彈 20% 時,即進入牛市。在完整資料集中,這產生了 27 個獨立的牛市/熊市區段(而非各一個長週期),合計 20,663 個牛市時段與 10,139 個熊市時段。

#### 按市場狀態與門檻劃分的存入與提領優勢,1 週時間跨度

| 門檻 | 存入,牛市 | 存入,熊市 | 提領,牛市 | 提領,熊市 |
|-----------|---------------|----------------|-------------------|--------------------|
| 100 萬美元以上 | +0.5% | **+3.9%** | -1.2% | **-3.9%** |
| 200 萬美元以上 | +0.7% | **+4.8%** | -1.8% | **-4.3%** |
| 500 萬美元以上 | +1.5% | **+4.9%** | -4.2% | **-6.1%** |
| 1,000 萬美元以上 | +2.2% | **+4.6%** | -3.1% | **-7.7%** |

在每一個測試規模下,存入優勢在熊市都比牛市更強,提領優勢在熊市都比牛市更差。這在四個獨立的規模切分下一致成立,而不只是基準案例,這種跨門檻的一致性,與本專案其他地方用來證明某個模式並非單一切分結果的方法相同。一個可能的解讀是:巨鯨在已經下跌的市場中賣出,反映了對進一步下跌的真實認知,而在牛市中賣出更多是例行的獲利了結,預測力較低。與此同時,提領訊號恰好在「買入信心」理應最強的時候,表現得更差。

在 24 小時與 3 天的時間跨度下,同樣的方向依然成立(熊市優勢的幅度在存入與提領上都一致大於牛市),但較弱,且顯著性較不一致。1 週時間跨度是這個模式最清晰的地方。三個時間跨度的完整結果見 `results/bull_bear_analysis.csv`。

這些都是未經校正觀測值重疊問題(詳見限制章節)的原始 p 值,因此,與本文件其他地方一樣,可能高估了統計信心。值得慶幸的是,由於時間跨度較短,獨立觀測值遠多於前述完全站不住腳的長時間跨度主張,因此這裡的高估程度應該較輕微,但並非零(詳見 `scripts/run_bull_bear_analysis.py`)。

狀態相依的可預測性在股票與商品市場中已獲得充分證實(Guidolin & Timmermann,2008;Haase & Neuenkirch,2023,後者發現可預測性集中在動盪時期,與本文發現的熊市優勢更強的結果一致)。目前尚未發現針對加密貨幣、以牛熊市狀態檢驗巨鯨訊號強度的文獻,因此本項結果是一項與上述文獻方向一致、但並非直接複現的新貢獻。也值得引述一個警示性的反例:Kirby(2023)指出,股票市場中部分牛熊市狀態轉換所呈現的「可預測性」,實際上可能只是報酬偏度(return skewness)造成的假象,而非真實的時變訊號,這種可能性在本文中同樣無法排除。

<a id="discussion"></a>
## 討論

### 為何提領優勢會衰退？(有證據支持)

在 2023 年,從交易所提領大多代表看漲的觀點。到了 2025 年,DeFi 已經成熟到人們會為了質押、提供流動性、橋接至 L2,或與各種協議互動而提領資金。這些行為都不具有價格方向性的資訊價值。

門檻分析支持了這個論點。如果問題只是小額參與者稀釋了訊號,那麼 1,000 萬美元以上的提領應該仍展現優勢。但這個訊號在每一個規模下都已失效,這指向的是提領行為意涵的結構性轉變。

優勢隨著愈來愈多參與者察覺而衰退,這是一個已獲充分記錄的普遍現象(McLean & Pontiff,2016,發現已發表的股市異常現象在樣本外與發表後分別損失 26% 至 58% 的報酬;Vukovic 等人,2026,則顯示加密貨幣市場效率本身也會隨時間變化)。而本文提出的具體機制——DeFi 成熟化正是改變提領行為意涵的原因——是本專案自己的假說:我們並未找到任何同行評審文獻直接證實,交易所提領行為愈來愈多代表質押、提供流動性或 L2 橋接,而非自我保管。這應被理解為一個合理、可檢驗的解釋,而非已獲證實的結論(詳見限制章節第 2 項)。

### 為何存入優勢得以存續並增強？

將 ETH 存入交易所的原因並沒有改變。這仍然代表你想賣出它。因此無論 DeFi 領域發生什麼變化,存入依然是乾淨的賣出訊號。

**一個比我們最初認為的更有力的論點。** 隨著 ETH 價格從 2023 年約 1,200 美元上漲到 2026 年的 4,000 美元以上,固定 100 萬美元的門檻捕捉到的參與者投入程度逐漸下降:2023 年約 833 顆 ETH,2026 年僅約 250 顆 ETH(詳見限制章節)。如果存入優勢只是單純的組成效應,稀釋理應隨時間削弱它,因為 100 萬美元以上的資金池會逐漸包含愈來愈多規模較小、較不精明的賣家。然而優勢反而增強了。優勢在本應不利於它的逆風下依然增強,這可能意味著存在真實的資訊優勢。

**優勢為何沒有被套利消除,這一點是推測,而非已獲證實的結論。** 一個假說是,巨鯨監控工具(Nansen、Arkham、Whale Alert)之所以不成比例地放大偏向買入訊號的活動,是因為那正是使用者想聽到的內容,使得賣出訊號較少被關注、較少被套利。我們沒有直接證據支持這個說法。簡短搜尋後找到一個間接、較無直接關聯的資料點:一項 2023 年的研究發現,比特幣價格對 Whale Alert 在 USDT 增發事件周圍發布的推文反應更強烈,而這類事件通常被解讀為看漲訊號。

我們能更有把握地說的是,優勢會隨交易規模擴大。極度貪婪期間 1,000 萬美元以上的存入,展現出資料集中最強的訊號。這些是在市場高點做出果斷行動的大型參與者。

### 巨鯨賣家的思考尺度是以月計算

存入優勢在 24 小時時很小(+1.3%)。在 1 個月時是 +4.8%。在 6 個月時是 +12.4%。這些巨鯨並非在做當沖交易。他們是在對未來一到兩季 ETH 的走向做出結構性判斷,而且他們對的機率高於隨機。在極度貪婪期間存入的巨鯨,6 個月後的平均報酬是 -22.6%。價格崩跌了,而巨鯨早已出場。

### 78.3% 這個主張發生了什麼事

我們最初報告,極度貪婪期間 1,000 萬美元以上的存入,正確預測 24 小時價格下跌的比例達 78.3%。仔細檢視這個數字的來源後發現,背後的交易集中在少數幾天內,全部發生在 2025 年的幾週之內,而非分散在整個資料集中。當一個統計數字背後的事件如此群聚時,它們大多是在描述同一批市場波動,而非許多各自獨立的事件。這正是限制章節所描述的觀測值重疊問題——在事件研究文獻中被正式定義為事件日期群聚所導致的橫截面相關性(cross-sectional dependence,Kolari & Pynnönen,2010)——也正是我們不信任這個特定數字的原因,儘管背後的交易與運算本身完全真實。

<a id="limitations"></a>
## 限制

1. **長時間跨度的觀測值彼此重疊。** 在 1 個月及以上的時間跨度,數千筆巨鯨事件實際上是在衡量同一次價格變動。二項式檢定的 p 值高估了顯著性。在這些時間跨度下,優勢欄位(巨鯨命中率減去基準率)比原始命中率或 p 值更具參考價值。討論章節中有一個具體案例,說明這個問題如何推翻了一個特定的頭條主張。這是事件研究文獻中一個已被正式定義的問題(即使橫截面相關性很低,事件日期的群聚仍會誇大表面上的顯著性:Kolari & Pynnönen,2010;後續研究將此推廣至部分重疊的事件視窗,這與隨時間跨度增長的優勢直接相關:Kolari、Pape & Pynnönen,2018),而非本專案特有的臨時性顧慮。學界標準的補救方法是日曆時間投資組合法(calendar-time portfolio method)或 Kolari-Pynnönen 校正檢定統計量;本專案並未採用這兩種方法,這正是為何我們將其揭露為限制,而非直接進行校正。

2. **DeFi 稀釋假說尚未經過驗證。** 我們推測提領優勢的衰退,是因為提領行為愈來愈多代表非方向性的 DeFi 活動。要驗證這一點,需要追蹤提領後的鏈上活動(例如,這筆 ETH 是流入質押合約,還是冷錢包？)。

3. **鏈上延遲。** 巨鯨交易在區塊確認後(約 12 秒)才可見,但監控、處理與執行回應交易還需要額外的延遲時間。

4. **固定美元門檻忽略了 ETH 價格的成長。** 2023 年的 100 萬美元交易約等於 833 顆 ETH,但到 2026 年僅約 250 顆 ETH。100 萬美元這個資金池會隨時間被規模較小的參與者稀釋。以 ETH 計價或經通膨調整的門檻會更嚴謹,但會使跨年度比較更困難。

5. **存入交易未經資金回轉(round-tripping)篩選。** 「巨鯨賣家的思考尺度是以月計算」這個框架,假設存入行為反映了審慎的長期觀點,但我們並不知道巨鯨為何存入資金。透過檢查同一地址是否隨後發生提領:24.1% 的存入,在 24 小時內見到同一地址發生提領;29.1% 在一週內發生,這與「持續數月的方向性觀點」這個假設,對至少一部分具有意義的事件而言並不一致。這項檢查僅確認是否發生了後續提領,並未核對金額是否與存入相符,因此無法區分真正的短期資金回轉與透過同一地址發生的不相關後續活動。整體的統計模式(存入先於下跌)也許依然成立,無論個別意圖為何,但「巨鯨在做出審慎判斷」這個敘事,應被理解為對整體現象的描述,而非對每一筆交易的斷言。

```bash
# 重新產生預先運算的資料(僅在底層資料變更時才需要執行)
python scripts/build_dashboard_data.py

# 啟動儀表板
streamlit run app/dashboard.py
```

儀表板共有七個區塊:按時間跨度劃分的存入優勢(1 小時至 6 個月)、年度穩定性(優勢衰退,可選擇時間跨度)、門檻敏感度(可選擇時間跨度)、以情緒為條件的命中率、報酬分佈的市場狀態探索器、按年度劃分的存入與提領不對稱性(可選擇時間跨度),以及限制章節。

儀表板在執行階段並不會讀取原始資料集。`scripts/build_dashboard_data.py` 會將所有顯示的數字預先運算並儲存至 `app/dashboard_data.json`(約 480 KB),而 `app/dashboard.py` 只讀取這個檔案。這使得儀表板能夠在不需要 187 MB 原始巨鯨資料集(已加入 .gitignore,從未提交)的情況下,部署到 Streamlit Community Cloud。

<a id="how-to-run"></a>
## 如何執行

```bash
# 安裝相依套件
pip install -r requirements.txt

# 核心分析:事件研究命中率
python scripts/run_event_study.py

# 逐年前向驗證
python scripts/run_walk_forward.py

# 門檻敏感度分析
python scripts/run_threshold_sensitivity.py

# 機器學習模型(前向驗證隨機森林)
python scripts/run_phase4_model.py

# 最大不利變動(持有期間的回撤)
python scripts/run_drawdown_analysis.py

# 巨鯨活動是否跟隨新聞情緒？
python scripts/run_sentiment_whale_consistency.py

# 按牛市/熊市市場狀態劃分的巨鯨訊號
python scripts/run_bull_bear_analysis.py

# 重新產生儀表板的預先運算資料,然後啟動儀表板
python scripts/build_dashboard_data.py
streamlit run app/dashboard.py
```

備註:原始巨鯨資料需要 Dune Analytics API 存取權限。由於檔案大小限制,預先處理的資料並未包含在儲存庫中。
